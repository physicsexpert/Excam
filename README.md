# Excam是一款基于openmv的多功能小相机
## Excam主控采用stm32h7343，搭载1.8寸180°翻转屏，电池，补光灯，RGB灯，按键，SD卡，支持OV5640、OV2640摄像头，兼容openmv固件（引出SWD接口），可以实现色块识别、人脸识别、边缘检测、模板匹配、特征点检测等功能，可以跑一些简单的神经网络模型。

# 结构说明

调试器外壳采用3D打印制作，总共需要打印五个部分：顶壳，底壳，镜头保护盖，开关，拓展模块，需要用电烙铁压入M2热熔螺母配合M2螺丝进行固定。
